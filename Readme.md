3️⃣ What is the difference between map(), filter(), and forEach()?

map()--->runs a function on every element of an array and returns a new array with the modified values. The original array is not changed

filter()--->runs a condition on every element of a array and returns only the elements or values which are similar with the filter

forEach()---> runs a function on every element of a array and loop throw them but did not return anything.
